# 🎯 FACE RECOGNITION WEB SYSTEM

A beautiful web application for face recognition, built with Flask and modern web technologies.

## ✨ Features

- 📊 Interactive dashboard
- ➕ Upload and manage face photos
- 🔍 Recognize faces in images
- 👥 Manage people database
- 🎨 Beautiful modern UI
- ⚡ Fast recognition (< 1 second)

## 🚀 Quick Start (3 Steps)

### Step 1: Download Files
All files are in this folder:
- `app.py` - Backend
- `requirements_web.txt` - Packages
- `templates/index.html` - Frontend
- `start.bat` or `start.sh` - Quick launcher

### Step 2: Run Setup

**Windows - Double-click:**
```
start.bat
```

**Mac/Linux - Run in terminal:**
```bash
chmod +x start.sh
./start.sh
```

### Step 3: Open Browser
```
http://localhost:5000
```

## 🎮 How to Use

1. **Add Faces Tab** - Upload photos of people
2. **Train System** - System learns the faces
3. **Recognize Tab** - Upload a group photo to identify people
4. **People Tab** - Manage your face database

## 🖥️ System Requirements

- Python 3.10+
- Windows/Mac/Linux
- 500MB free space
- Modern web browser
